# GaiaNexus

## Installation

```bash
# Install GaiaNode
curl -sSfL 'https://github.com/GaiaNet-AI/gaianet-node/raw/refs/heads/feat-integrate-nexus/install.sh' | bash

# Move the wasm file to the GaiaNode directory
mv llama-api-server.wasm $HOME/gaianet/
```

## Usage

```bash
# initialize the node
gaianet init

# start the node
gaianet start

# stop the node
gaianet stop
```
